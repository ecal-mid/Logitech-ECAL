This is a distance-based musical instrument with simple controls:

  1. Start Recording: Right-click to start recording mode
  2. Drop Notes: Left-click to manually place notes as you move
  3. Play Mode: Right-click to reset distance and enter playback mode

Additional Controls:

  • Long Left Press: Reset sequence and start new recording (clears all notes)
  • Right-click + drag: Restart the sequence (resets total distance to 0 without clearing notes)

Notes are saved with their distance from the start and replay when you pass through similar distances in playback mode.
